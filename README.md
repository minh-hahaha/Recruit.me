# Recruit.me (Team Maryland)

Link:
http://recruit-me3.s3-website-us-east-1.amazonaws.com/

Use Cases implemented for Iteration 2:

Company:
  - Activate Job
  - Close Job
  - Review Applicants for job
  - Offer Jobs
  - rescind offer to applicant

Applicant:
  - Search Jobs [page]
  - Apply to job
  - accept offer
  - withdraw from job
  - reject offer from company

Company:
  - Register Company
  - Review Company Profile
  - Edit Company Profile
  - Create Job
  - Edit Job

Applicant:
  - Register Applicant
  - Review Profile
  - Edit Profile
